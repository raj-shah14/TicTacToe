***Steps to run the file***
1)Open tictactoe.py in Python 3.X
2)Under the "MAIN FUNCTION" section, change the board_size parameter to have different dimension board.
3)Run the code
4)Follow further instructions in the code

NOTE: When prompted to enter Next Move, Enter the move with row number and column number separated by space. 
For example if we want to get the following output,

  1 2 3
 +-+-+-+
1|X| | |
 +-+-+-+
2| | | |
 +-+-+-+
3| | | |
 +-+-+-+

Enter as shown
Next move
1 1
